case study/strategic portfolio Project

python ( pandas, metplotlib, seaborn, numpy)

The goal of this Project is to analyed all AirBnb Listing 
in New York in the year 2024, uncover the distribution pattern across
the various listing in New York.
## STEPS
Importing all necessary libraries
Pandas for data structure and DataFrames
Numpy for numerical computing, multi-dimensional arrays
Matplotlib for creating static and interactive visuals
Seaborn for high-level visualization such as heatmap, regression plots
